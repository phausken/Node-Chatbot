## Setup
Clone the repo:
```
$ git clone https://github.com/shinetext/se-test-template
```

Install the dependencies:
```
$ npm install
```

Run the server:
```
$ npm start

or

$ node server.js
```

View it in your browser at: http://localhost:3000
